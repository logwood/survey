| 论文名称与发表年份/出处 | 核心机制（一句话总结它的技术亮点） |
| :---- | :---- |
| **D-REX** (ICLR 2026\) | 构建可微的 Real-to-Sim-to-Real 引擎，从无标注视频中自动推断提取目标物体质量等动态参数，进而渲染出携带精准力反馈约束的高保真灵巧抓取训练数据。 |
| **RISE** (arXiv 2026\) | 将 RL 数据探索完全转移至“组合式世界模型”中，通过连续生成多视角物理想象轨迹并进行内部价值纠错，实现零硬件成本的策略自我生成与扩容。 |
| **MeshMimic** (arXiv 2026\) | 结合 3D 视觉基础模型，从无位姿标注的消费级单目视频中联合提取人体运动轨迹与底层地形网格，自动合成针对复杂地形耦合的物理交互监督数据。 |
| **Fail2Progress** (CoRL 2025\) | 针对真实机器人执行失败的案例，利用斯坦变分推断（SVI）在并行仿真中全自动定向合成海量具有高变异性的边界困难场景数据，用以高效修复策略漏洞。 |
| **RAYNOVA** (CVPR 2026\) | 摒弃显式 3D 先验，采用基于 Plücker 射线位置编码的时空自回归架构，能在任意未见视角和极端相机运动下合成极致保真度的具身物理视频数据。 |
| **OptimSyn** (ICLR 2026\) | 运用基于目标模型梯度的影响函数机制引导奖励模型，训练合成数据评估引擎（Rubrics），从而在算法层级自动筛选并生成最能提升下游任务的合成训练集。 |
| **ParticleFormer** (CoRL 2025\) | 突破纯像素限制，利用基于时空 Transformer 的粒子化动态机制对三维点云进行特征预测，自动合成具备多物体、多材质深度物理连贯性的三维轨迹数据。 |
| **DynaRetarget** (arXiv 2026\) | 融合逆向运动学提取与采样轨迹优化（SBTO），能直接将高度复杂的真实人类动态操作录像重定向并提炼为符合零样本迁移需求的动力学可行机器人数据集。 |
| **Sim2Real-VLA** (ICLR 2026\) | 集成场景投影、域随机化与技能获取的全自动数据合成引擎，能在纯合成数据上训练支持零样本真机迁移的双系统VLA。 |
| **RoboGene** (arXiv 2026\) | 构建基于“最少使用（LFU）”采样与反馈自反思的多智能体框架，全自动生成海量且物理可执行的长尾机械臂操作任务数据。 |
| **Real2Render2Real (R2R2R)** (CoRL 2025\) | 完全摒弃动力学仿真，纯粹利用3D高斯重建与球面线性插值运动学渲染，将单一真实视频裂变为数千条高质量的跨构型训练轨迹。 |
| **RoboSplat** (RSS 2025\) | 提出在统一的3D高斯表征空间中直接进行场景替换、等变操作与视角变换，通过一次真实演示衍生出海量多模态数据。 |
| **Mirage2Matter** (arXiv 2026\) | 从多视角视频中提取物理接地的3DGS基底并结合运动规划器，在重构的隐式世界模型中合成逼真的视觉动作配对数据。 |
| **EgoHumanoid** (arXiv 2026\) | 设计视角转换与动作空间对齐的系统性映射管线，突破性地将大规模第一人称人类演示视频提炼为双臂移动机器人的高维训练数据。 |
| **ConverSeg-Net** (ICRA 2026\) | 构建了完全无需人类干预的 AI 闭环数据引擎，专门用于大批量合成具身空间推理所需的强语义 Prompt-Mask（对话式提示与掩码）配对数据。 |
| **DynScene** (CVPR 2025\) | 提出统一的扩散合成框架，通过自然语言端到端地联合生成符合四元数量化物理稳定性的静态场景布局与无碰撞的动态操作轨迹数据。 |
| **UltraDexGrasp** (ICRA 2026\) | 将优化驱动的抓取位姿综合与基于规划的演示生成深度融合，自动化构建了包含2000万帧、适配上千种物品的双臂多策略合成数据集。 |
| **ROPA** (ICRA 2026\) | 引入骨架位姿条件微调扩散模型，在保证物理接触约束与几何一致性的前提下，离线合成具有全新视角的第三人称RGB-D数据及动作标签。 |
| **One View, Many Worlds** (CoRL 2025\) | 利用大模型生成的域随机化资产克服模态鸿沟，从单张参考图像自动合成涵盖海量视角与光照的6D位姿估计算法训练集。 |
| **OmniRetarget** (ICRA 2026\) | 基于“交互网格”拉普拉斯变形引擎，在严格保留人、物与环境接触关系下将人类视频重定向为无穿模的机器人训练数据。 |
| **DexMan** (arXiv 2025\) | 从无标定的单目RGB野生视频中提取高密度3D点轨迹线索，通过残差强化学习自动转化为多指人形机器人的灵巧操作训练数据。 |
| **Seed2Scale** (ICRA 2026\) | 采用小模型高速采集、大模型校验的协同机制，构建了仅需 4 个初始演示便能持续自我演进与扩容高质量数据的具身引擎。 |
| **RoboCurate** (ICRA 2026\) | 针对由视频生成模型产出的带有噪点的合成动作，创新性引入VLM物理属性对齐机制，实现了对残缺生成数据的智能剔除与策展。 |
| **LodeStar** (CoRL 2025\) | 采用VLM智能切分人类长序列演示，并在仿真中通过残差RL对脆弱瓶颈动作进行局部强化与探索，自动规模化扩增鲁棒连接轨迹数据。 |
| **CUPID** (CoRL 2025\) | 在自动数据生成管线末端首创“影响函数”机制，通过评估单条合成轨迹对模型策略性能的具体梯度贡献来智能策展并剔除损害性噪声样本。 |
| **KALIE** (ICRA 2025\) | 利用预训练视觉语言模型的物理点位先验建立“可供性感知”合成管道，通过提取人类零星演示在想象环境中自动化派生海量高质量抓取数据。 |
| **Track2Act** (ECCV 2024 / arXiv 2025\) | 采用独立分离时间步长的联合视频-动作扩散模型，直接从无标注互联网视频中预测稠密3D点云轨迹提取广义运动规律以合成泛化动作数据。 |
| **Constraint-Preserving Data Gen** (CoRL 2025\) | 在生成和扩增单样本监督轨迹的过程中显式融合运动学边界与接触动力学约束，确保系统合成出的视觉运动策略数据安全且物理可行。 |
| **Action-View Augmentation** (CoRL 2025\) | 利用新颖视点合成（NVS）结合操作级轨迹扭曲变形，在保证运动学连贯性的条件下，将单一专家演示自动化衍生出一千条高变异训练轨迹。 |
| **HybridGen** (arXiv 2025\) | 借助VLM将复杂操作任务智能解耦为依赖专家的局部交互与可自动规划的空间移动，通过局部矩阵变换结合混合规划大幅裂变有限初始演示。 |
| **ReinforceGen** (arXiv 2025\) | 将任务分解、数据生成与运动规划相结合形成初始演示解决方案，随后通过在线强化学习试错微调来自动化生成并优化多阶段技能数据。 |
| **Self-Improving VLA (PLD)** (ICLR 2026\) | 在基础VLA策略推演时让残差RL专家主动接管环境，全自动生成带有纠错和恢复行为的高质量混合轨迹数据，专门反哺微调下游VLA大模型。 |
| **DreamGen** (CoRL 2025\) | 运用基于动作编码控制的视频大模型在连续隐空间内仿真环境状态转移，为训练跨场景机器人策略生成具备丰富长尾属性的预演数据集。 |
| **PhysVLM** (CVPR 2025\) | 建立包含密集物理和几何特征的指令微调合成数据引擎，使得基础多模态模型能够跨域深刻理解并判断机器人在复杂空间中的物理可达性。 |
| **GraspMolmo** (CoRL 2025\) | 在大规模平行环境仿真中结合多模态大语言模型，批量生成极具多样性的合成视觉与语言对齐数据，增强开放几何抓取泛化。 |
| **Terminal-Task-Gen** (arXiv 2026\) | 在沙盒化容器环境中通过教师智能体构建强一致性的多轮验证网络，利用自动化运行的测试用例将自由文本合成为逻辑缜密的强化控制数据。 |
| **RoboSimGS** (RA-L / arXiv 2025\) | 提供从纯视觉输入到高保真物理世界还原的完整“Real2Sim2Real”映射管道，通过3DGS渲染生成带真实纹理反馈的可交互数据。 |
| **Re³Sim** (arXiv 2025\) | 将高斯渲染环境与基于网格的底层几何推理解耦融合，建立不受限于单一光照和物体材质的光度真实感仿真环境用以低成本扩张数据。 |
| **DreamDojo** (ICLR 2026\) | 直接从海量第一与第三人称互联网野生视频中逆向提取泛化的交互动态特征，于隐空间中合成物理合理的世界模型预训练底层先验数据。 |
| **dARt Vinci** (arXiv 2025\) | 专门为医学手术控制构建的可扩展视觉数据采集系统，通过高度解耦的自我中心机制大规模录制微操作级别的动作映射数据。 |
| **PointBridge** (ICRA 2026\) | 提出域不可知的点级隐式表征自动提取机制，彻底打破视觉模态壁垒，使得仅靠纯几何合成数据即可训练出高泛化的真实世界策略。 |
| **DexFlyWheel** (NeurIPS 2025\) | 结合模仿学习与仿真环境残差强化学习的主动探索试错，构建自动回收高成功率接触交互轨迹并扩充数据集的自我演进飞轮。 |
