| 论文名称与发表年份/出处 | 核心机制（一句话总结它的技术亮点） |
| :---- | :---- |
| **Real2Render2Real (R2R2R)** (CoRL 2025\) | 摒弃了复杂的刚体动力学仿真，纯粹利用 3D 高斯溅射与运动学并行渲染，通过球面线性插值将单一真实人类视频自动裂变为数千个高质量的跨构型机器人训练轨迹。 |
| **DynScene** (CVPR 2025\) | 提出基于扩散模型的统一合成框架，通过自然语言指令端到端地联合生成符合四元数量化物理稳定性约束的静态场景布局与无碰撞操作轨迹。 |
| **Ctrl-World** (arXiv/ICLR 2025\) | 利用大语言模型动态泛化任务指令并重置环境初始状态，引导生成式世界模型在隐空间内自动合成呈海量多样性的后训练（Post-training）策略轨迹。 |
| **DexMan** (arXiv 2025\) | 突破性地从无相机标定、无深度的纯野生第三方单目 RGB 视频中，利用三维点轨迹线索恢复人体运动，并借由残差强化学习转化为多指人形机器人的灵巧操作数据。 |
| **RoboWheel** (arXiv 2025\) | 利用循环变分自编码器（Cycle VAE）实现跨实体的双向子空间对齐，将真实世界复杂的人类手-物交互（HOI）演示直接转化为无缝适配多构型的机器人闭环数据。 |
| **FlowVLA** (arXiv 2025\) | 在自回归预测中引入“视觉思维链”（先预测光流，后渲染外观）的因果机制，强制要求基础模型在合成未来多帧数据时必须遵守严格的时空物理一致性。 |
| **RoboScape** (arXiv 2025\) | 将 RGB 视频生成、时序深度预测与自适应关键点跟踪集成于联合学习的自回归框架，通过隐式注入几何与材质先验，生成具有高度物理可信度的世界模型数据。 |
| **Robot-Powered Data Flywheel (Scanford)** (arXiv 2025\) | 部署基础模型驱动的机器人在复杂开放环境中自主执行任务并记录长尾失败边界样本，构建自动收集高价值困难数据与策略持续微调的物理闭环飞轮。 |
| **Track2Act** (ECCV 2024\) | 通过在联合模型中采用完全独立分离的扩散时间步长来预测互联网视频中的三维点云轨迹，在消除模态重叠表征的同时提取广义运动规律以实现零样本操作。 |
| **NovaFlow** (arXiv 2025\) | 借助视频生成大模型与单目深度网络提取运动流规律，蒸馏出不依赖特定夹爪构型的“可执行的 3D 物体流”，实现从单一视觉观察到连续控制动作的流转化。 |
| **Constraint-Preserving Data Gen** (CoRL 2025\) | 在生成和扩增单样本（One-Shot）监督轨迹的过程中显式融合运动学和力学接触物理边界，确保生成的视觉运动策略数据安全且严苛遵循物理约束。 |
| **HybridGen** (arXiv 2025\) | 利用大视觉语言模型（VLM）将复杂模仿学习任务智能解耦为“强依赖专家的局部交互”与“可自动求解的空间移动”，通过局部矩阵变换结合混合规划实现数据规模化。 |
| **CUPID** (RSS 2025\) | 首创基于统计梯度的“影响函数”（Influence Functions）机制作为自动数据策展的核心，通过评估合成轨迹对策略性能的直接梯度贡献来智能剔除损害性噪声样本。 |
| **RoboGene** (arXiv 2026\) | 构建集成多环节反思能力的闭环智能体框架，利用最低频使用（LFU）多样性采样策略和长期记忆人类反馈模块，剔除幻觉指令并全自动生成物理可行的操作长尾任务集。 |
| **SpatialVLM** (CVPR 2024\) | 开发全自动空间 VQA 引擎生成高达 20 亿规模的精确度量空间（Metric Space）三维几何推理问答样本，大幅赋能基础模型的零样本定量物理估计能力。 |
| **PhysVLM** (CVPR 2025\) | 利用包含高密度物理属性的合成数据进行指令微调，使得基础多模态模型能够跨域深刻理解机器人在复杂环境中的物理可达性（Physical Reachability）。 |
| **ReKep** (CoRL 2025\) | 提出通过在空间三维域与时域内对关联关键点（Relational Keypoint）网络进行形式化数学约束推理，将自然语义指令自动转化为可供底层优化的数值型动作数据集。 |
| **WorldVLA** (arXiv 2025\) | 将图像、动作分词统一于单一自回归框架，通过创新的动作掩码机制预测物理世界的视觉演进，利用生成的“虚拟闭环反馈”加强动作决策流形的泛化鲁棒性。 |
| **DexFlyWheel** (arXiv 2025\) | 将遥操作模仿学习与仿真环境中的残差强化学习（Residual RL）结合，构建了一个自动利用环境试错来扩容高成功率接触交互轨迹的灵巧操作策略飞轮。 |
| **OmniH2O** (CoRL 2024\) | 建立了通用的从广义人类运动视频到复杂双足人形机器安全控制命令的重定向映射机制，支持利用野生数据集端到端地输出高自由度全身遥操作训练信号。 |
| **TSIA (Geometrically-Aware)** (arXiv 2025\) | 引入几何感知的任务空间模仿算法机制，仅需单次演示即可自动合成遵循复杂操作边界且轨迹平滑的几何路径，实现向未见空间分布的稳定零次训练迁移。 |
| **RAM (Retrieval-based Affordance)** (CoRL 2024\) | 开发检索驱动的可供性（Affordance）迁移框架，通过在海量数据先验库中匹配几何抓取特征，自动合成适用于新类别对象的通用操控策略轨迹。 |
| **FP3** (arXiv 2025\) | 利用超大规模、涵盖海量点云观察序列的多样化三维交互空间数据进行扩散 Transformer 预训练，构筑了可通过极少量微调便能生成各类下游精确任务的 3D 基础策略层。 |
| **Gen2Act** (arXiv 2024\) | 创新性地在条件视频生成模型上施加闭环执行结构，在预测新场景下的人类操作动作后提取视觉反馈流，将其精确映射为机器人跨模态操控数据。 |
| **EC-Flow** (arXiv 2025\) | 提出以目标图像为硬条件的光流对齐机制，并与 URDF 模型参数解析深度融合，使得深度生成的抽象视觉流向量能直接被逆解析为附带物理执行边界的具体关节驱动力矩。 |
| **RIGVid** (arXiv 2025\) | 在处理自由文本指令时同步预测带有真实空间尺度的动态演化深度图，并运用物体刚体位姿跟踪技术反解出极具置信度的 6DoF 回放运动参数以供底层直接跟随。 |
| **Robots Pre-train Robots** (ICLR 2025\) | 应用大型离线特征蒸馏算法，将不同物理规格与观测空间的异构机器人轨迹统一投影至以操作为核心的隐式预训练特征空间，自动生成高维度控制监督表征。 |
| **OpenBot-Fleet** (arXiv 2024\) | 创建基于众包智能手机感知终端的低成本移动车队云框架，通过异步通信和边缘计算，在多变的室外环境中全自动采集并迭代处理百万级的群体动态避障与导航数据。 |
| **DART / DexHub** (arXiv 2024\) | 整合了增强现实（AR）渲染映射与极低延迟云端状态同步架构，彻底突破了传统遥操作的认知疲劳极限，实现了高质量人类操作序列的高通量（High-throughput）去中心化记录。 |
| **Terminal-Task-Gen** (arXiv 2026\) | 在沙盒化（Sandboxed）容器环境中构建强一致性的多轮验证教师智能体，通过嵌入自动化运行的验证测试用例，将开放文本合成为逻辑缜密且100%保真的强化控制数据闭环。 |
