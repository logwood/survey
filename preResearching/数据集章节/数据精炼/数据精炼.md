# Data Engine 相关工作贡献梳理

> 说明：以下文档按四个小小节组织，聚焦每篇工作的**核心贡献**、**适合放置的位置**与**简短评价**，方便后续筛选进综述正文。

## 1. Visual and Viewpoint Augmentation

### GenAug

* **核心贡献**：将预训练图像-文本生成模型引入机器人模仿学习，进行语义一致的视觉增强，在不改变任务本质的前提下重绘场景、物体与背景。
* **价值**：证明了来自大规模互联网视觉先验的生成模型，能够在少量真实机器人数据下显著提升对新场景和新物体的泛化。
* **适合定位**：这一方向的早期代表，适合作为“生成式视觉增强”的起点。

### VISTA (View-Invariant Policy Learning via Zero-Shot Novel View Synthesis)

* **核心贡献**：提出基于 zero-shot novel view synthesis 的视角增强框架，用单视角示范合成未见相机视角下的观测，以学习 view-invariant policy。
* **价值**：把“新视角生成”明确纳入策略学习训练环节，而不只是作为视觉模型任务；特别适合支撑“视角不变性”这条叙事。
* **适合定位**：适合作为 2024 年视角增强代表作。

### RoVi-Aug

* **核心贡献**：联合进行 robot appearance augmentation 与 viewpoint augmentation，把同一示范转换为不同机器人外观和不同相机配置下的数据。
* **价值**：把视觉增强从“场景/物体变化”扩展到“机器人本体与相机布置变化”，更直接服务 cross-embodiment 与 cross-camera 泛化。
* **适合定位**：视觉增强段中承上启下的一篇，连接场景增强与 embodiment augmentation。

### RoCoDA

* **核心贡献**：把 causal invariance、SE(3) equivariance 与 counterfactual augmentation 统一到一个框架中，系统性地修改任务无关因素并同步调整动作。
* **价值**：相比经验式增强，RoCoDA 提供了更“原则化”的数据增强视角，强调何种扰动不应改变策略，何种几何变换应诱导动作等变。
* **适合定位**：既可放在视觉增强，也可作为过渡到“物理与几何约束增强”的桥梁。

### RoboEngine

* **核心贡献**：提出 plug-and-play 视觉机器人数据增强工具链，提供语义机器人分割、背景生成和 task-aware scene generation，使用户能够低门槛生成 physics- and task-aware 的增强场景。
* **价值**：与单篇方法不同，RoboEngine 更像“增强基础设施”，强调无需相机标定、绿幕等苛刻前提即可进行高质量增强。
* **适合定位**：这一小节中最适合作为“augmentation toolkit / engine”代表。

### ROPA

* **核心贡献**：通过 pose-conditioned 生成机制，为第三人称 RGB / RGB-D 双臂操作数据合成新的机器人姿态与对应动作标签。
* **价值**：从单纯视觉改图像进一步走向“带标签的可控增强”，同时兼顾接触约束与几何一致性。
* **适合定位**：适合放在视觉增强小节后半段，作为向“视觉+动作联合生成”过渡的代表作。

### H2R (Human-to-Robot Data Augmentation for Robot Pre-training)

* **核心贡献**：把人类视频或人类操作观测转换为更接近机器人视角和机器人训练分布的数据源，用于机器人预训练。
* **价值**：拓宽了机器人可利用的数据来源，不再局限于机器人自身采集的 demonstration。
* **适合定位**：适合作为“跨观测域/跨主体数据迁移增强”的补充文献。

### EgoMimic

* **核心贡献**：构建从 egocentric human video 到机器人模仿学习的数据管线，使第一人称人类操作视频成为大规模机器人预训练信号。
* **价值**：虽然不完全是传统 augmentation，但它从源头上扩大了训练数据来源，适合放在视觉来源扩展的语境中。
* **适合定位**：可作为本小节末尾或引向 synthetic data generation 的过渡文献。

## 2. Synthetic Demonstration and Trajectory Generation

### Real2Render2Real (R2R2R)

* **核心贡献**：提出不依赖动力学仿真或机器人硬件的训练数据生成流程，只需 smartphone 扫描和单个 human demo，即可生成大量 robot-agnostic manipulation 数据。
* **价值**：显著降低了数据生成门槛，把“数据扩增”升级为“从极少真实输入中系统化生成训练集”。
* **适合定位**：这一小节中非常强的代表，可用来定义“synthetic demonstration generation”这条线。

### DreamGen

* **核心贡献**：利用 video world models 生成 photorealistic synthetic robot videos，再通过 latent action model 或 inverse dynamics model 恢复 pseudo-actions，形成可训练的 neural trajectories。
* **价值**：把机器人数据扩展从手工或规则增强推进到“由生成式世界模型直接合成训练轨迹”的范式。
* **适合定位**：适合作为 2025 年以后“world-model-driven data engine”的代表作。

### RoboSplat (Novel Demonstration Generation with Gaussian Splatting Enables Robust One-Shot Manipulation)

* **核心贡献**：利用 3D Gaussian Splatting 重建真实操作场景，并直接在 3D 表示上编辑物体、视角、光照、外观和 embodiment，生成新 demonstrations。
* **价值**：相比 2D 图像增强，它显著提高了跨视角与跨场景的一致性；相比经典仿真，它更贴近真实视觉几何。
* **适合定位**：适合作为“3D representation–based demonstration generation”代表。

### CyberDemo

* **核心贡献**：在仿真中采集人类 demonstrations，并在模拟器内做大规模 augmentation，再用少量真实数据微调，实现 dexterous manipulation 的 sim-to-real 迁移。
* **价值**：把“人类示范 + 模拟器增强 + 少量真实适配”串成了完整闭环，对灵巧操作尤其有代表性。
* **适合定位**：适合作为从 simulated human data 到 real-world policy 的一条支线。

### Action-View Augmentation / One Demo is Worth a Thousand Trajectories

* **核心贡献**：从单条 expert demonstration 自动生成大量高方差训练轨迹，同时联合做 novel view synthesis 与 manipulation-level trajectory warping。
* **价值**：这篇工作的亮点在于视觉增强与动作轨迹增强是同步进行的，而非只改观测不改控制信号。
* **适合定位**：可放本节，也可与下一节“Physically Grounded”交叉提及。

## 3. Physically Grounded and Recovery-Aware Augmentation

### DMD (Diffusion Meets DAgger)

* **核心贡献**：用 diffusion model 合成偏离 expert demonstrations 的 OOD / failure states 下的观测，从而在不额外昂贵收集 DAgger 数据的情况下补充 recovery supervision。
* **价值**：把“恢复数据生成”纳入 augmentation 范式，解决模仿学习在分布外状态下易崩溃的问题。
* **适合定位**：这一节的 recovery-aware augmentation 代表作。

### CP-Gen (Constraint-Preserving Data Generation)

* **核心贡献**：从单条 expert trajectory 出发，通过关键点轨迹约束、几何变换、关节优化和 collision-free motion planning 生成新的 physically feasible demonstrations。
* **价值**：强调“扩增不只是变一变轨迹，而是必须保持几何与技能约束”，因此非常适合 tight-tolerance 和多阶段操作任务。
* **适合定位**：这一节最强的“constraint-preserving augmentation”代表之一。

### Physics-Driven Data Generation for Contact-Rich Manipulation via Trajectory Optimization

* **核心贡献**：整合 physics-based simulation、VR human demonstrations、retargeting 与 trajectory optimization，自动生成大规模动态可行的 contact-rich manipulation 数据。
* **价值**：相比简单的几何增强，它显式确保接触过程、动力学一致性和跨 embodiment 迁移的可行性。
* **适合定位**：适合作为 contact-rich / planning-guided data generation 的代表文献。

### RESample

* **核心贡献**：针对 VLA / IL 数据中缺乏失败与恢复样本的问题，利用离线 RL 价值网络与 exploratory sampling 自动发现并加入 OOD bottleneck states 与 recovery-critical segments。
* **价值**：这篇工作把 recovery augmentation 做得更系统，不再只依赖单次扩散生成，而是通过探索式采样自动扩充错误恢复数据。
* **适合定位**：适合放在 DMD 后面，体现从“生成纠偏样本”到“主动挖掘恢复关键状态”的演进。

### Planning-Guided Diffusion Policy Learning for Generalizable Contact-Rich Bimanual Manipulation

* **核心贡献**：把 planning 或 model-based reasoning 引入策略学习与数据生成过程，以提升复杂接触双臂操作中的泛化与物理可执行性。
* **价值**：这类工作说明 augmentation 不一定只来自生成模型，也可以来自规划器和轨迹优化器的结构性先验。
* **适合定位**：作为这一节的补充文献，用于丰富“planning-guided augmentation / data generation”视角。

## 4. Data Curation, Reweighting, and Verification

### L2D (Learning to Discern)

* **核心贡献**：针对质量和风格不一致的人类 demonstrations，学习一个能泛化到新 demonstrators 的质量评估器，并据此进行更稳健的 imitation learning。
* **价值**：把“学会辨别什么是好 demonstration”显式引入训练流程，是数据筛选线的重要起点。
* **适合定位**：这一小节的开篇文献很合适。

### Re-Mix

* **核心贡献**：在大规模机器人数据预训练中，用 distributionally robust optimization 学习不同数据域/机器人域的 mixture weights，以优化最坏域表现。
* **价值**：强调问题不只是“删哪些数据”，还包括“不同数据该占多大权重”，对 foundation-policy 训练尤其关键。
* **适合定位**：非常适合作为“reweighting / mixture optimization”代表作。

### SCIZOR

* **核心贡献**：在 transition 粒度上进行 self-supervised curation，同时过滤 suboptimal state-action pairs 和冗余样本；具体通过任务进度预测去除低质量片段，通过 joint state-action deduplication 去重。
* **价值**：相比 trajectory-level 筛选，它更细粒度，也更适合大规模 foundation-style imitation datasets。
* **适合定位**：这一小节中最强的“transition-level curation”代表。

### CUPID

* **核心贡献**：基于 influence functions 估计每条 demonstration 对闭环期望回报的影响，从而实现有原则的过滤旧数据与选择新数据。
* **价值**：把数据价值评估从启发式提升到“对策略性能的因果影响”层面，是非常值得写进综述的一篇。
* **适合定位**：适合作为 influence-based data curation 的代表作。

### DataMIL

* **核心贡献**：利用 datamodels 做端到端、policy-aware 的数据选择，不靠语义相似度或人工质量直觉，而是直接选择最能提升目标任务表现的数据。
* **价值**：强调“好数据”是相对于当前任务和当前策略定义的，代表更任务导向的数据选择路线。
* **适合定位**：可与 CUPID 并列，形成 influence-based 与 datamodel-based 两类 policy-aware curation 视角。

### Robot Data Curation with Mutual Information Estimators

* **核心贡献**：利用互信息估计每条 demonstration 对整个数据集 state-action 统计结构的贡献，同时刻画状态多样性与动作可预测性。
* **价值**：提供了信息论角度的数据质量衡量方式，可用于按“信息价值”而非单纯任务标签来筛选 demonstrations。
* **适合定位**：适合作为信息论路线的数据筛选补充。

### RoboCurate

* **核心贡献**：提出一个 synthetic robot data generation + verification 框架，一方面用 controllable video generation 增强多样性，另一方面通过 simulator replay 与生成视频的运动一致性来验证动作质量、过滤低质量样本。
* **价值**：它把“生成”和“验证”绑定在一起，避免只用 VLM 或视频外观 plausibility 来判断样本质量。
* **适合定位**：这一小节末尾非常合适，可作为 synthetic data verification 代表作。

### OXE-AugE

* **核心贡献**：针对 OXE 数据集 embodiment 分布失衡的问题，构建大规模机器人增强数据集与 AugE-Toolkit，把 OXE 扩展到更多机器人本体组合和数百万新轨迹。
* **价值**：说明数据引擎不只是 sample-level augmentation，也可以是 dataset-scale augmentation，用于推动 cross-embodiment policy learning。
* **适合定位**：可放在本小节结尾，作为“dataset scaling / large-scale augmentation infrastructure”代表。

## 一句话归纳（可直接用于综述写作）

* **Visual and Viewpoint Augmentation**：通过改变外观、视角、主体或观测域来扩大 perception-side distribution support。
* **Synthetic Demonstration and Trajectory Generation**：从少量示范、扫描或世界模型中直接生成新的 demonstrations、video-action pairs 或 full trajectories。
* **Physically Grounded and Recovery-Aware Augmentation**：在扩增数据时显式维护几何、接触、动力学与恢复行为的可执行性。
* **Data Curation, Reweighting, and Verification**：通过筛选、加权、去重和验证，提高大规模异构数据的有效密度与下游策略性能。
